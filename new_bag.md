  WebSocket Proxy — как это работает

  Текущая архитектура (проблема):

  ┌─────────────┐      ┌─────────────┐      ┌─────────────────┐
  │  Браузер    │ ──── │   Vercel    │ ──── │   ElevenLabs    │
  │ (Беларусь)  │      │    (US)     │      │      API        │
  └─────────────┘      └─────────────┘      └─────────────────┘
         │                                          │
         │  1. GET /api/signed-url ────────────────►│
         │  2. ◄──── signed_url (wss://...) ────────│
         │                                          │
         │  3. WebSocket НАПРЯМУЮ к ElevenLabs ────►│
         │     IP = Беларусь = БЛОК! ❌              │
         └──────────────────────────────────────────┘

  С WebSocket Proxy:

  ┌─────────────┐      ┌─────────────────┐      ┌─────────────────┐
  │  Браузер    │ ──── │  WebSocket      │ ──── │   ElevenLabs    │
  │ (Беларусь)  │      │  Proxy (US/EU)  │      │      API        │
  └─────────────┘      └─────────────────┘      └─────────────────┘
         │                     │                        │
         │  WebSocket ────────►│                        │
         │  к прокси           │  WebSocket ───────────►│

         │  ◄─────────────────────────────────────────►│
